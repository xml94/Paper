# Slide Hotkeys

* Ctrl+D=Ctrl+C + Ctrl+V
* Ctrl+G, Ctrl + Shift + G
* Ctrl+Alt+V:为了保持内容格式的一致，选择性粘贴也是我们常用的一种快捷键之一
* Ctrl+【 或 】缩放文字
* F4,重复上一步
* Ctrl+鼠标拖动对象=复制
* Ctrl+Shift+鼠标拖动对象=水平或垂直方向复制
* 选中对象，按住Shift，拖动对象 = 垂直/平行移动
* 选中对象（图形或图片），按住Shift，拖动尺寸控点 = 比例缩放
* Ctrl + Shirt + 拖动尺寸控点=中心&比例缩放
* Ctrl+E=居中
* Alt+F10=选择窗格
